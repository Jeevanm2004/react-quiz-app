# React Quiz App

A simple quiz application built using React.

## WEBSITE-Demo
[WEBSITE-Demo](https://jeevanm2004.github.io/react-quiz-app/)

## Features
- Multiple choice questions
- Timer-based quiz
- Score calculation
- High score tracking

## Technologies Used
- React
- JavaScript
- CSS

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Deployment
This project is deployed using GitHub Pages.
